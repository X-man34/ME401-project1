### LaTeX Report Template
This folder contains a latex formatted text file which can be used to generate the final technical report for project 1. Students are encouraged to change the report as they see fit to meet their project specifications, but the level of technical rigor and conceptual completeness should be considered baseline.

also, included is a pdf of the report template output.

### Recommended Use:

Copy the text from the .txt file and paste it directly into the overleaf LaTeX editor.
